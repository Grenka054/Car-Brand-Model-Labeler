Справка

Добро пожаловать в раздел помощи для программы исправления разметки марок и моделей. Здесь вы найдете инструкции по использованию различных функций программы.

Инструкции

  • Загрузить CSV: Загрузить файл CSV, содержащий данные о марках и моделях.
  • Сохранить CSV: Сохранить текущий файл CSV после внесения изменений.
  • Предыдущий ID: Перейти к предыдущему ID для просмотра и редактирования данных.
  • Следующий ID: Перейти к следующему ID для просмотра и редактирования данных.
  • Удалить изображение: Пометить текущее изображение как удаленное. Изображение будет исключено из текущего списка, но не удалено из файла CSV.
  • Копировать изображение: Скопировать текущее изображение в буфер обмена для дальнейшего использования.
  • Исключить отрицательные ID: Отфильтровать отрицательные ID из текущего списка, чтобы сосредоточиться на положительных значениях.

Горячие клавиши

  • Ctrl+S: Сохранить текущий файл CSV.
  • Ctrl+Z: Отменить последнее действие.
  • Ctrl+Y: Повторить последнее отмененное действие.
  • Ctrl+C: Скопировать текущее изображение.
  • Del: Пометить текущее изображение как удаленное.

Часто задаваемые вопросы

  • Как загрузить новый файл CSV? Воспользуйтесь меню "Файл" и выберите пункт "Загрузить CSV".
  • Как сохранить изменения? Используйте меню "Файл" и выберите пункт "Сохранить CSV" или нажмите Ctrl+S.
  • Что делать, если данные не отображаются корректно? Убедитесь, что файл CSV содержит все необходимые столбцы и данные правильно отформатированы.
  • Как восстановить удаленные данные? Используйте функцию отмены действия (Ctrl+Z) для восстановления последних изменений.
